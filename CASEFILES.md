# Purpose

The Case File is the fundamental unit of knowledge within Project 250.

Everything else exists to support it.

Artifacts begin investigations.

Case Files preserve them.

Every historical investigation, regardless of subject, follows the same structure.

Consistency enables trust.

---

# Definition

A Case File is a structured, evidence-based investigation documenting the complete known history of a historical artifact.

A Case File is never static.

It evolves as new evidence emerges.

Every revision is preserved.

---

# Core Philosophy

A Case File should answer one question:

**"Can another investigator independently reach the same conclusion?"**

If the answer is no—

the Case File is incomplete.

# Standard Structure

## 1. Cover

Case Number

Title

Investigation Status

Lead Investigator

Date Opened

Last Updated

Version

---

## 2. Executive Summary

A concise overview of:

The artifact

Its significance

Major findings

Outstanding questions

Current confidence

---

## 3. Artifact Profile

Artifact Type

Title

Date

Creator

Dimensions

Material

Language

Current Owner

Condition

Repository

Accession Number (if applicable)

---

## 4. Investigation Question

Every Case File begins with one primary question.

Examples

Who owned this book?

Is this inscription authentic?

Can the provenance be reconstructed?

What historical events connect to this artifact?

---

## 5. Evidence

Title

Description

Evidence Type

Source

Date

Confidence

Attachments

Relationships

Notes

Photographs

Documents

Maps

Registers

Receipts

OCR

Digital Images

Published Sources

Statement

Supporting Evidence

Counter Evidence

Confidence

Status

Supported

Disproven

## 11. Open Questions

Investigations remain open.

Every unresolved question should be documented.

---

## 12. Research Leads

Potential records

Potential experts

Potential collections

Priority

# Success Criteria

A Case File succeeds when:

Another investigator can reproduce the investigation.

Every conclusion is traceable.

Every source is cited.

Every assumption is identified.

Every uncertainty is documented.

Trust is increased.
